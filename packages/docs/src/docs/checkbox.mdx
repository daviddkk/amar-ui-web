## Basic Usage

You can also click on the label to tick the checkbox. Be sure to add `id` in the input and `for`
attribute in th label associating the checkbox.

```html preview=true
<div class="Checkbox MarginBottom">
  <input class="Checkbox-control" id="checkboxExample1" type="checkbox" />
  <label class="Checkbox-label FontWeight-medium" for="checkboxExample1">
    Check me!
  </label>
</div>
<div class="Checkbox">
  <input class="Checkbox-control" id="checkboxExample2" type="checkbox" />
  <label class="Checkbox-label FontWeight-medium" for="checkboxExample2">
    Check me too!
  </label>
</div>
```
