## Usage

Bottom bars enables users to navigate through main pages of our application. Each bottom bar item
is associated with an icon to visually represent its destination and often accompanied with a text label.

```html preview=true bgColor=lighter size=small
<div class="BottomBar">
  <ul class="BottomBar-menu">
    <li class="BottomBar-menuItem">
      <a href="#">Pinjamanku</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Riwayat</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Bantuan</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Lainnya</a>
    </li>
  </ul>
</div>
```

## States

### Active State

In order to activate the active state of a bottom bar item, simply append `.is-active` state inside
of `BottomBar-menuItem`.

```html preview=true bgColor=lighter size=small
<div class="BottomBar">
  <ul class="BottomBar-menu">
    <li class="BottomBar-menuItem">
      <a href="#">Pinjamanku</a>
    </li>
    <li class="BottomBar-menuItem">
      <a href="#">Riwayat</a>
    </li>
    <li class="BottomBar-menuItem is-active">
      <a href="#">Lainnya</a>
    </li>
  </ul>
</div>
```
