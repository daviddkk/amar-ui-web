## Text Color

Use `.Color-<theme>` class to change the **text color** of an element.

```html preview=true
<div class="Color-primary">Primary Color</div>
<div class="Color-secondary">Secondary Color</div>
<div class="Color-tertiary">Tertiary Color</div>
<div class="Color-danger">Danger Color</div>
<div class="Color-warning">Warning Color</div>
<div class="Color-info">Info Color</div>
<div class="Color-light BgColor-dark">Light Color</div>
<div class="Color-dark">Dark Color</div>
```

## Background Color

Use `.BgColor-<theme>` class to change the **background color** of an element.

```html preview=true
<div class="BgColor-primary Color-white">Primary Color</div>
<div class="BgColor-secondary Color-white">Secondary Color</div>
<div class="BgColor-tertiary Color-white">Tertiary Color</div>
<div class="BgColor-danger Color-white">Danger Color</div>
<div class="BgColor-warning Color-black">Warning Color</div>
<div class="BgColor-info Color-white">Info Color</div>
<div class="BgColor-light Color-black">Light Color</div>
<div class="BgColor-dark Color-white">Dark Color</div>
```

## Using mixins

```scss
.SomeComponent {
  color: tnk-color-theme('primary'); // switches text color to primary theme.
  background-color: tnk-color-theme('primary'); // switches background color to primary theme.
}
```
