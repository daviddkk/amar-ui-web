## Via Variables

If you need to customize some of a component's specs, please try to find the matching variables
before opting to hardcode its styles directly.

For example, assume we want to adjust Card's default spacing. You could try the solution below:

```scss
// before
$tnk-card-spacing: tnk-spacing('base');

// after
$tnk-card-spacing: tnk-spacing('large');
```

## Override styles directly

Whenever possible avoid this solution.

For example:

```scss
.Card {
  // before
  padding: $tnk-card-spacing;

  // after
  padding: tnk-spacing('large');
}
```
